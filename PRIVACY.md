# 隐私政策 / Privacy Policy

最后更新 / Last updated: 2026-05-30

## 中文

**图片下载器（Image Downloader）扩展不收集、不存储、不传输任何用户数据。**

- 本扩展仅在你**主动点击工具栏图标**时，读取**当前标签页**中的图片地址和尺寸，用于在弹窗中展示列表。
- 所有处理都在你的浏览器本地完成。扩展**不会**将任何数据发送到任何服务器，也不包含任何分析、统计或追踪代码。
- 下载图片时，由浏览器内置的下载功能直接从图片原始地址获取文件，扩展不经手图片内容。
- 扩展不读取浏览历史、cookie、表单、账号或任何个人信息。

简言之：没有数据离开你的设备。

如有疑问，可在项目仓库提交 issue：
https://github.com/zed1902209846-dotcom/image-downloader-extension

## English

**The Image Downloader extension does not collect, store, or transmit any user data.**

- The extension only reads image URLs and dimensions from the **current tab**, and only when you **actively click the toolbar icon**, in order to display the list in its popup.
- All processing happens locally in your browser. The extension does **not** send any data to any server and contains no analytics, telemetry, or tracking code.
- When downloading, the browser's built-in download feature fetches the file directly from the image's original URL; the extension never handles image contents itself.
- The extension does not read your browsing history, cookies, form data, accounts, or any personal information.

In short: no data ever leaves your device.
